About this site

Here you will find some of my recent work in web page and app design. In 2021, I started learning front end development. I've learned HTML, CSS, JavaScript, and React. I've done some work in backend using Python. Prior to that, and ongoing, I've been studying machine learning and data engineering.

For about 20 years I designed surface modeling algorithms for computer-aided design systems. This included writing analytic and numerical surface intersectors and surface evaluators, trimming curves and recalculating surface boundaries, implementing Boolean operations on solid models with intersection curve chaining and coincidence analysis, filleting and rounding edges, and more.

I enjoy every new challenge. I also love all kinds of puzzles, playing the piano from Bach to Debussy, and making colorful arts and crafts.

Original JavaScript and React Apps

Cryptogram Assistant Github repository

I like solving cryptograms on paper, and I prefer to use a pen rather than a pencil. So either I get it right from the start, or I have an inky mess. So when a friend suggested I write a game, I immediately thought of a helpful formatted cryptogram app that lets me try out different solutions. There are a couple dozen cryptograms included. While researching other cryptogram sites, I noticed some of them were quite addictive. My goal in writing this was to make a non-addictive game. Therefore, entering the guess letters requires some work, there's no prize at the end, and the number of provided quotes is limited. But it's still a lot of fun. Written in React. More

Palette Assistant Github repository

I was motivated to write this app after I saw a similar grid of colors sorted alphabetically, and then sorted naively by red first, then green, then blue. I thought I could do a better job. I soon learned that sorting colors is a thorny problem. Visual media companies that need to determine color brightness apply different weights to the red, green and blue components of light to compute a lightness value. Initially I allowed the user to rearrange the color chips and recompute the weights, but my friends and family preferred to choose palettes, so that's what this app does. Written in React. More

Penrose Laboratory Github repository

With this app I'm reliving my glory days as a student of electrical engineering. This career consisted of exactly two semesters of Laboratory Electronics at the Harvard Extension School. During the second semester, we built a computer on a breadboard with chips and wires using the Motorola 68004 microprocessor. I wrote a program for this computer to generate Penrose tiling patterns on an oscilloscope display. At first I entered machine language instructions directly onto a programmable RAM. When things got complicated, I switched to an assembly language translator. This is basically the same algorithm, except the language is React and the output is to the browser. More
